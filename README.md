# Restaurant Menu– Restaurant Website

## 📌 Project Overview
Restaurant Menu is a responsive restaurant website designed to showcase food menu items, restaurant information, and allow customers to easily contact the restaurant or reserve a table.

The project focuses on creating a clean and interactive user interface that improves the online presence of a restaurant and enhances the user browsing experience.

This project demonstrates fundamental frontend web development skills using HTML, CSS, and JavaScript.


##  Features

- Interactive restaurant menu section
- Multiple food categories (Pizza, Salads, Starters)
- Landing page with restaurant introduction
- About section with restaurant details
- Contact section with reservation form
- Responsive design for different screen sizes
- Clean and simple user interface

## Website Sections

The website includes the following sections:

- Home – Landing page with restaurant introduction  
- Menu – Display of food items and categories  
- About – Information about the restaurant  
- Contact – Reservation and contact form
